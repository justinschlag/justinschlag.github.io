Welcome to JustinBot

JustinBot is a conversational AI assistant designed to answer questions about my experience, projects, and research in an approachable and informative way. Whether you're a recruiter, a student, or someone simply curious about my work, JustinBot is here to help.

No technical skills are needed—just type your question and JustinBot will generate a response using natural language. This makes it perfect for anyone who prefers quick summaries over reading pages of documentation or portfolios.

How JustinBot Works – A Deep Dive

JustinBot isn’t just a chatbot—it's a full-stack, interactive application combining real-world deployment, machine learning, and dynamic frontend development. Here’s how the system comes together:

  1. Assistant API (the AI Brain): JustinBot leverages OpenAI’s Assistant API, which provides responses based on a finely tuned language model. This model has been trained on a wide variety of topics and is further specialized to represent my background and experiences.
  2. Node.js Server (the Middleman): The frontend doesn’t connect directly to OpenAI. Instead, a Node.js server acts as a secure intermediary. When you ask a question, the request is sent to my backend, which reformats it, sends it to the Assistant API, then returns the result as a clean reply.
  3. Hosted on Render: This server is hosted on Render, a scalable platform that keeps the service low-cost but always available. Because it scales dynamically, the first request may take up to 30 seconds—hence the sleeping message.
  4. Frontend Chat Interface (the Presentation Layer): Built using HTML, CSS, and JavaScript, the UI is responsive, styled professionally, and dynamically updates content without page reloads. It’s designed to be fast, accessible, and mobile-friendly.

Why this matters: JustinBot demonstrates both the ability to deploy real-world AI solutions and to explain technical work clearly. It combines backend logic, frontend engineering, deployment infrastructure, and conversational UX—key skills in modern engineering.
