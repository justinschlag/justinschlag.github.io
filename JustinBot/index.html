JustinBot is powered by OpenAI's Assistant API and runs on a Node.js server hosted with Render. It's a custom AI assistant trained with info about my work, personality, and interests. Here's how it all works:

  • Assistant API: The brain behind the bot! I created a personalized assistant using OpenAI's platform, which handles responses to questions in a conversational way.
  • Node.js Backend: When you submit a question, a Node.js server sends your message to OpenAI, waits for the reply, then delivers it back to the site.
  • Render Hosting: The backend is hosted on Render. It spins up when needed, which is why JustinBot might take a few seconds to "wake up."
  • Frontend Chat: The chatbox on this page is built with plain HTML, CSS, and JavaScript. It fetches answers in real-time and updates the page without reloading.
